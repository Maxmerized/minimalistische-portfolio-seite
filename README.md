# Gustav Dressler Portfolio

Eine minimalistische, elegante Portfolio-Website für Gustav Dressler.

## Über die Website

Diese Portfolio-Website präsentiert die Werke und Projekte von Gustav Dressler, einschließlich:

- **Märchen** 
- Zwei veröffentlichte Bücher über Gier, Gerechtigkeit und Weisheit
- 
**Chroniken** - Historische Dokumentationen über Sitzenberg-Reidling
- 
**Schweizerhof** - Bio-Reitstall mit 70 Hektar pestizidfreier Fläche
- 
**MIËLO** - Partnerschaft mit dem österreichischen Honig-Projekt
- 
**Presse** - Medienberichte und Publikationen
- 
**CV** - Lebenslauf und beruflicher Werdegang


## Technologien

- HTML5
- CSS3 (mit CSS Grid & Flexbox)
- Vanilla JavaScript
- Formspree (für Kontaktformular)

## Installation & Nutzung

1. **Repository klonen:**
   ```bash
   git clone https://github.com/IHR-USERNAME/gustav-dressler-portfolio.git
   cd gustav-dressler-portfolio
   ```



## Projektstruktur

```
gustav-dressler-portfolio/
├── index.html              # Hauptdatei mit allen Seiten
├── styles.css              # Komplettes Styling
├── script.js               # Navigation & Interaktionen
├── README.md              # Diese Datei
├── .gitignore             # Git-Konfiguration
└── [Bilder]               # Alle Portfolio-Bilder
```

## Deployment

Die Website kann kostenlos gehostet werden auf:

- **GitHub Pages** (empfohlen für GitHub-Nutzer)
- **Vercel** (schnell, mit GitHub-Integration)


## Anpassungen

Alle Inhalte können einfach angepasst werden:

- **Texte ändern:** Bearbeiten in `index.html`
- **Farben anpassen:** Ändern der CSS-Variablen in `styles.css`
- **Bilder austauschen:** Bilddateien im Hauptordner tauschen
                           (und im code umbenennen)
